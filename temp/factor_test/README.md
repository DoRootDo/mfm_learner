## 因子筛选

- 这个因子是不是符合基本的标准，用这个标准，直接过滤掉不合格的因子（目前不考虑变废为宝的利用）
- 合格因子，应该可以排个序，那么这个这个排序是什么标准？
- 不同的时期的测试的因子，是否有出入？用的股票池不同，是否有出入？
